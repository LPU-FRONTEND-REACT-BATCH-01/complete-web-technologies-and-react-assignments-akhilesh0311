Box Sizing Property

Both boxes have `width: 300px`, `padding: 20px`, and `border: 5px solid`.

content-box (Default)

Actual width = 300 + (20*2) + (5*2) = 350px

This box expands beyond 300px because padding and border are added to the width.

border-box

Actual width = 300px (padding and border cut into the width)

This box stays exactly 300px wide. Padding and border are included in the width calculation!
