# HTML Day 1: Introduction to HTML
## Practice
1. Create a basic HTML document structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`).
2. Add a `<title>` to your webpage.
3. Practice with headings `<h1>` to `<h6>`.
4. Create paragraphs `<p>`.
5. Add links using `<a>`.
6. Add images using `<img>`.

## Assignments
- Create a simple "About Me" page using headings, paragraphs, a picture of yourself (or a placeholder), and a link to your favorite website.
