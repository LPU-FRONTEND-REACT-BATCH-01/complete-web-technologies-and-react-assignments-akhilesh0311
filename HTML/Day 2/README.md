# HTML Day 2: Lists, Tables, and Basic Formatting
## Practice
1. Create Ordered Lists `<ol>` and Unordered Lists `<ul>` with List Items `<li>`.
2. Build a Table `<table>`, `<tr>`, `<th>`, `<td>`.
3. Understand Block vs Inline elements (e.g., `<div>` vs `<span>`).
4. Formatting text with `<strong>`, `<em>`, `<b>`, `<i>`.

## Assignments
- Create a page with your weekly schedule using a table.
- Below the schedule, create a list of your hobbies (unordered) and top 3 favorite movies (ordered).
