# HTML Day 4: Semantic Layouts
## Practice
1. Use semantic HTML5 layout tags instead of just `<div>`.
2. Learn `<header>`, `<footer>`, `<main>`, `<section>`, `<article>`, `<aside>`, and `<nav>`.

## Assignments
- Restructure a basic webpage using only semantic tags. Create a header with a nav, a main area with an article and aside, and a footer.
