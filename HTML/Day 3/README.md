# HTML Day 3: Exploring Forms
## Practice
1. Create a form element `<form>`.
2. Add various input types (`text`, `email`, `password`, `radio`, `checkbox`, `date`).
3. Use `<label>` for inputs.
4. Add a `<textarea>` for multiline input.
5. Create dropdowns with `<select>` and `<option>`.
6. Add a submit `<button>`.

## Assignments
- Create a Contact Us form or a Registration form. Ensure all inputs are properly labeled.
