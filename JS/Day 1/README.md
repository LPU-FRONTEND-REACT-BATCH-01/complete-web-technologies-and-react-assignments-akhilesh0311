# JS Day 1: Basics, Variables and Data Types
## Practice
1. Linking a `.js` script to an HTML file.
2. Logging to console (`console.log`, `console.error`).
3. Variables: `var`, `let`, `const`.
4. Primitive Data Types: Strings, Numbers, Booleans, Null, Undefined.
5. Basic Math Operators (`+`, `-`, `*`, `/`, `%`).

## Assignments
- Create a script that defines some variables about you (name, age, isStudent) and prints a formatted string to the console using template literals (backticks).
