# JS Day 2: Control Flow and Intro to Arrays
## Practice
1. Conditionals (`if`, `else if`, `else`).
2. Comparison Operators (`==`, `===`, `>`, `<`, `!=`).
3. Loops: `for` loop, `while` loop.
4. Arrays introduction.
5. Array basic methods: accessing elements, `.length`, `.push()`, `.pop()`.
